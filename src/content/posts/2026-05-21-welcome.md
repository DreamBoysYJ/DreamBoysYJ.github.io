Velog에서 글을 편하게 쓰고, 최종 원본은 GitHub Markdown으로 남긴다.

여기서부터 이 블로그는 한글 글의 기준점이 된다. Medium에는 영어판을 올리고, Tistory나 SNS에는 필요할 때 맞춰 변환한다.

> 문체, 농담, 욕설은 글쓴이의 일부다. 기술적으로 틀린 것만 조심스럽게 고친다.

```js
const flow = [
  "write on Velog",
  "save Markdown on GitHub",
  "review technical claims",
  "localize for Medium",
  "publish after approval"
];
```

첫 글은 작게 시작한다. 자동화는 나중에 더 붙이면 되고, 중요한 건 원본을 잃지 않는 것이다.
